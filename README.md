# Img2txt_V1 -- Windows
=======================

>Ceci est la première version de **Img2txt**, le convertisseur d'image en pgm et texte !
Cette première version fonctionne avec une interface en commandes *(pour la version graphique, consultez à partir de la version 3.0)*.


## Fonctionnalités
  - Conversion d'une image pnj/jpg en PGM
  - Utilisation et changement de "palettes" de texte pour convertir des .pgm en fichier texte 
  - **Palettes** :

```
1. Défaut "W", "w", "l", "i", ":", ",", ".", " "
```
```
2. Étendue "@", "#", "S", "%", "?", "*", "+", ";", ":", ",", ".", "`", " "
```
```
3. Blocs "█", "▓", "▒", "░", " ", " "
```
```
4. Nature_lumiere "♠", "♣", "♥", "♦", "◊", "○", "●", "◌", " ", " "
```
```
5. Détails_fins "@", "#", "S", "%", "*", "+", "=", ":", "-", ".", " "
```
```
6. Ombre_lumiere "▇", "▆", "▅", "▃", "▂", "▁", "▀", "·", " ", " "
```
```
7. Cyberpunk "■", "□", "◉", "○", "◍", "◎", "●", "◌", " ", " "
```
```
8. Medieval "☠", "♔", "♖", "♗", "♘", "♙", "⚔", "⛓", " ", " "
```
```
9. Aquarelle "♫", "☼", "☁", "☂", "☺", "♀", "♂", "♻", " ", " " 
```
```
10. Haute_def "@", "▓", "▒", "░", "■", "●", "◘", "○", "◙", "◌", "◍", "☼", "☺", "♫", "♦", "♠", "♣", "♥", "◊", "♂","♀", "∴", "∵", "≈", "≠", "≡", "≣", "⊕", "⊗", " "
```
```
11. Gradients"██", "▛▛", "▙▙", "▜▜", "▟▟", "▚▚", "▞▞", "▗▖", "▄▄", "▀▀", "▂▂", "▃▃", "▅▅", "▆▆", "▇▇", "■□", "●◌", "◉○", "◍◎", " "

```   
  - Inversions du rendu
  - Redimensionnement de la taille du rendu
  - Affichage dans la console
  - Enregistrement dans un fichier texte

## Comment l'utiliser ?
  - Lancez l'executable : en ligne de commande pour linux, double clique pour Windows
  - Renseignez un fichier image ou PGM qui est dans le même dossier que l'exécutable OU son chemin 
  - Jouez avec les paramètres de palettes et de redimensionnement pour avoir le résultat souhaité
  - Enregistrez !

## Installation
  - Téléchargez le .exe joint à cette release 
  - Décompressez
  - Fini !
